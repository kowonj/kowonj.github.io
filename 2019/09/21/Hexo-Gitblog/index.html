Git Repository

Create two repositories

You need two repositories:

  1. Repo for framework:
    a. Create a repo (private) ex. myBlog
    b. git clone the repo to local
    c. Then, branch off from master

    • This is the directory you need to install Hexo framework.


  2. Repo for publishing posts:
    a. Create a repo. You need set the name of the repo as below. The repo should be ‘public’.



Hexo

Install Hexo

  • a. Install Hexo-cli

  • b. Install Hexo: If you are already in the directory you want to install Hexo, you don’t need to enter the dir for init command.


  • If you are not in the directory, then specify the folder name.
